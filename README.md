# 📚 LearnPath AI

> AI-powered **personalized learning path generator** using knowledge graphs + collaborative filtering

[![Python](https://img.shields.io/badge/Python-3.9+-blue)](https://python.org)
[![AI](https://img.shields.io/badge/AI-Adaptive%20Learning-purple)](.)
[![Graph](https://img.shields.io/badge/Knowledge-Graph-orange)](.)

## 🧠 How It Works

```
Learner Profile (skills, goals, speed)
        ↓
Knowledge Graph (topic prerequisites DAG)
        ↓
Collaborative Filtering (similar learner patterns)
        ↓
Adaptive Path Generation
        ↓
Personalized Weekly Schedule
```

## 🚀 Quick Start

```bash
python learnpath.py
```
